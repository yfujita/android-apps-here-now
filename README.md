# Here Now

Current location dashboard app with a Sci-Fi aesthetic.

## Features
- **Location Info**: Displays current Address, Latitude, and Longitude.
- **Environmental Data**: Shows real-time Elevation and Gravitational Acceleration.
- **Nearest Station**: Auto-detects and displays the closest train station (HeartRails Express).
- **Map Integration**: One-tap access to external maps.
- **UI**: Futuristic, high-density interface with glassmorphism effects.

## Technologies
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM
- **Network**: Retrofit

## Setup
1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle and run on your device.

## APIs
- [GSI (Geospatial Information Authority of Japan)](https://maps.gsi.go.jp/development/elevation_s.html) - Elevation data
- [HeartRails Express](http://express.heartrails.com/) - Station data
